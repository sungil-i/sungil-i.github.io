# 수업에 대한 개괄적인 오리엔테이션

## 교재

- NCS교재와 함께 보충자료를 제공합니다
- 프린트물 + 웹페이지(모바일)

## 배울 내용

- (학습상황) Java 기초 완료
- HTML/CSS/자바스크립트 일부 이해
- C# 을 배우고 Unity 를 할 수 있다.

##  응용 프로그램 만들기

- 게임과 비슷한 반응형 응용 프로그램을 만듭니다
- 코딩(프로그래밍 언어)
- 웹 브라우저에서 사용할 수 있는 반응형 응용 프로그램을 먼저 만들보면서, 게임 프로그래밍에 대한 개념을 잡습니다
- 유니티를 사용하는 기초적인 방법도 배웁니다.
